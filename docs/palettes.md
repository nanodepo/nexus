# Themes & Palettes

Theme palettes live in `src/config/nexus.php` under `themes`. Each theme defines the same 7 **base colors**, and the system expands them into the 14-color ordered palette used by the UI.

## Active Theme
```php
'theme' => env('NEXUS_THEME', 'default')
```

## Theme Structure
```php
'themes' => [
    'default' => [
        'handler' => DefaultTheme::class,
        'colors' => [
            'red' => '#ff0000',
            'orange' => '#ffa500',
            'yellow' => '#ffff00',
            'green' => '#00ff00',
            'aqua' => '#00ffff',
            'blue' => '#0000ff',
            'purple' => '#ff00ff',
        ],
        'target_lightness' => 0.60,
        'max_chroma' => 0.16,
        'chroma_scale' => 0.65,
    ],
    'gruvbox' => [
        'handler' => GruvboxTheme::class,
        'colors' => [
            'red' => '#cc241d',
            // ...
        ],
    ],
    'catppuccin' => [
        'handler' => CatppuccinTheme::class,
        'colors' => [
            'red' => '#d20f39',
            // ...
        ],
    ],
    'nord' => [
        'handler' => NordTheme::class,
        'colors' => [
            'red' => '#bf616a',
            // ...
        ],
    ],
],
```

## How 14 Colors Are Built
1. **Base colors** are defined for:
   `red, orange, yellow, green, aqua, blue, purple`
2. **Intermediates** are generated by OKLCH interpolation between neighbors:
   `red-orange, orange-yellow, yellow-green, green-aqua, aqua-blue, blue-purple, purple-red`
3. The final ordered palette is:
   `red → red-orange → orange → orange-yellow → yellow → yellow-green → green → green-aqua → aqua → aqua-blue → blue → blue-purple → purple → purple-red`

## Gradient Defaults
```php
'gradient' => [
    'from' => env('NEXUS_GRADIENT_FROM', 'red'),
    'to' => env('NEXUS_GRADIENT_TO', 'purple'),
],
```

## Usage
```php
$colors = gradient(6);              // uses defaults above
$colors = gradient(6, 'red', 'blue');
$colors = gradient(6, '#cc241d', '#458588');
```

## Override in App
To customize, override `nexus.theme` (select another) or replace `nexus.themes` with your own.

To inspect the final 14-color palette at runtime:
```php
\NanoDepo\Nexus\Theme\ThemeRegistry::colors('catppuccin');
```
